Git Git fue lanzado públicamente por Linus Torvalds el 7 de abril de 2005. Git fue creado por Linus Torvalds, quien también es conocido por ser el creador del kernel de Linux. Mi conocimiento tiene una fecha de corte en septiembre de 2021 Git está principalmente desarrollado en el lenguaje de programación C, lo que le proporciona una alta eficiencia y rendimiento. Git es un ejemplo de software libre y de código abierto. Se distribuye bajo la Licencia Pública General de GNU versión 2 (GPLv2)
Mercurial Mercurial, también conocido como "hg" (el nombre del comando de línea para Mercurial), fue iniciado en 2005, pero su primera versión pública fue lanzada en marzo de 2005. Mercurial fue creado principalmente por Matt Mackall. Al igual que con Git, mi conocimiento tiene una fecha de corte en septiembre de 2021, por lo que no tengo información actualizada sobre la última versión estable de Mercurial que haya sido lanzada después de esa fecha. Te recomendaría visitar el sitio web oficial de Mercurial o fuentes confiables para obtener la información más reciente. Mercurial está principalmente desarrollado en el lenguaje de programación Python. Mercurial es un software libre y de código abierto distribuido bajo la Licencia Pública General de GNU versión 2 (GPLv2).
Apache Subversion(SVN) La primera versión pública de Apache Subversion (SVN) fue lanzada el 17 de octubre de 2000. SVN fue originalmente desarrollado por CollabNet, Inc., fundada por Brian Behlendorf, pero ha sido mantenido y desarrollado por una comunidad de voluntarios a lo largo de los años. Al igual que en casos anteriores, mi conocimiento tiene una fecha de corte en septiembre de 2021, por lo que te recomendaría consultar el sitio web oficial de Apache Subversion o fuentes confiables para obtener la información más actualizada sobre la última versión estable. Apache Subversion está principalmente desarrollado en el lenguaje de programación C. Apache Subversion es un proyecto de software libre y de código abierto bajo la Apache License.
